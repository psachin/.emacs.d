** Custom functions
   - Surround string with tags
     #+BEGIN_SRC emacs-lisp
       (defun surround(tag)
         "Surround word within TAG.
         TAG can be <>," ",' ',[ ], etc."
         (called-interactively-p "sWord should be inside: ")
         (backward-word)
         (mark-word)
         (when (region-active-p)
           (kill-region (region-beginning) (region-end)))
         (insert tag)
         (backward-char)
         (yank))
     #+END_SRC
